FAQs

Berapa lama proses pembuatannya kak?

1 – 3 hari selesai, tergantung paket undangan, revisi dan data yang kamu berikan.

Bagaimana saya mendapatkan data bukutamu?

Untuk data kehadiran tamu, Admin akan kirimkan link Google Sheet yang akan terupdate secara otomatis.

Apakah saya bisa request design?

Bisa banget, silahkan diskusikan dengan Admin mengenai design yang kamu mau.

Proses Order

Proses Order

  1. Proses pengerjaan akan dilakukan setelah Full Payment diterima
  2. Pembayaran dilakukan ke rekening yang sudah di infokan
  3. Konfirmasi pembayaran melalui WhatsApp Admin
  4. Mengisi form data undangan (untuk kelengkapan data dan file foto bisa menyusul)
  5. Proses pengerjaan akan dimulai
  6. Ketika Undanganmu jadi, kamu akan dikabarkan oleh Admin melalui WhatsApp
  7. Selanjutnya adalah proses revisi
  8. Link Undangan siap disebarkan
